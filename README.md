# PandaPerWorldSeed

## Configuring

config/per_world_seed.json generated at runtime

```
{
  "OVERWORLD": -2226254867561344606,
  "THE_NETHER": 5127212450885126531,
  "THE_END": 7103252580855421531
}
```
Replace with the seeds you want to use for each world
